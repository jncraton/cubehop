Simple 3D platformer game built with Three.js. Players navigate platforms to reach a goal.

[Demo](https://jncraton.github.io/cubehop/)

## Usage

1. Run make to install dependencies
2. Open index.html in a browser
3. Use WASD to move and Space to jump

## Testing

Run make test
